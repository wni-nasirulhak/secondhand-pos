====================================================
     E-COMMERCE LIVE SCRAPER - HOW TO START
====================================================

QUICK START (EASIEST WAY):
--------------------------
1. Double-click: START-SIMPLE.bat
2. Wait 2-3 seconds
3. Browser will open automatically!

STOP SERVER:
-----------
Double-click: STOP-SIMPLE.bat

ALTERNATIVE FILES:
-----------------
If START-SIMPLE.bat doesn't work, try:
- scripts\start.bat (with logs)
- scripts\start.ps1 (PowerShell)
- scripts\START-SILENT.vbs (no window)

TROUBLESHOOTING:
---------------

   '====' is not recognized...
   
Use: START-SIMPLE.bat (no special characters!)
