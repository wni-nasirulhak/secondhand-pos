# 🚀 Quick Start - StorageState Mode

## ทำตามนี้เลย! (3 ขั้นตอน)

### 1️⃣ ติดตั้ง Dependencies ใหม่

```bash
cd "All Project/tiktok-live-scraper"
npm install
```

---

### 2️⃣ Restart Server

```bash
Ctrl+C
npm start
```

---

### 3️⃣ ใช้งาน

1. เปิด http://localhost:3000

2. เลือก:
   ```
   🔐 โหมด Authentication: StorageState (แนะนำ!)
   ```

3. ติ๊ก:
   ```
   ☑️ 🥷 Stealth Mode
   ```

4. **ปิด Headless Mode** (ครั้งแรก)

5. กดเริ่ม

6. **Login TikTok ด้วยตัวเอง** ใน browser ที่เปิดขึ้นมา

7. Login สำเร็จ → Scraper บันทึก cookies

8. **ครั้งต่อไปไม่ต้อง login!** เปิด Headless Mode ได้

---

## ✅ ผลลัพธ์

- ✅ Login ครั้งเดียว
- ✅ Cookies ถูกบันทึกไว้
- ✅ Incognito Mode + มี login
- ✅ Stealth (ป้องกัน bot detection)
- ✅ ไม่โดน IP block
