# 🚀 แก้ IP Block ด้วย Chrome Profile (3 ขั้นตอน!)

## ✅ แก้ปัญหา IP Block แบบฟรี ไม่ต้องใช้ Proxy!

---

## ขั้นตอนที่ 1: หา Chrome Path

**ดับเบิลคลิก:**
```
find-chrome-profile.bat
```

**คัดลอก path ที่แสดง เช่น:**
```
C:\Users\YourName\AppData\Local\Google\Chrome\User Data
```

---

## ขั้นตอนที่ 2: ปิด Chrome ทั้งหมด

**⚠️ สำคัญมาก!**

1. กด `Ctrl+Shift+Esc` (Task Manager)
2. หา `chrome.exe`
3. End Task ทั้งหมด

---

## ขั้นตอนที่ 3: ตั้งค่าใน UI

1. **Restart Server:**
   ```bash
   Ctrl+C
   npm start
   ```

2. **เปิด UI:** http://localhost:3000

3. **ติ๊ก:**
   ```
   ☑️ 🌐 ใช้ Chrome Profile ที่ Login ไว้แล้ว
   ```

4. **กด "🔍 หาอัตโนมัติ"** หรือวาง path

5. **กดเริ่ม!**

---

## 🎉 เสร็จแล้ว!

- ✅ Chrome เปิดพร้อม login TikTok อยู่แล้ว
- ✅ ไม่โดน IP block
- ✅ ฟรี!

---

## 🐛 ถ้ามี Error

### "Profile is in use"
→ ปิด Chrome ให้หมดใน Task Manager

### "Cannot find profile"
→ เช็ค path ใน Chrome: `chrome://version`
